# Project H - Setup Instructions

To run this platform on your new laptop, follow these steps:

### 1. Install Node.js
If you don't have Node.js installed, download it from [nodejs.org](https://nodejs.org/). I recommend the **LTS (Long Term Support)** version.

### 2. Copy the Files
Copy the `website_files` folder to your USB and then to your new laptop.

### 3. Automatic Setup
Open the `required_tools` folder and double-click the `setup.bat` file. This will automatically install all the necessary tools and libraries needed for the website to run.

### 4. Run the Website
Once the setup is done, you can run the website using:
- **Backend**: `cd website_files/backend && npm start`
- **Frontend**: `cd website_files/frontend && npm run dev`

### 5. Config
You can edit `website_files/config.json` at any time to change the AI's name or the company name without needing to touch the code!
